# Retro Brutalist Portfolio

A modern, high-performance portfolio website with a "classical" retro/brutalist aesthetic. Built with pure HTML, Vanilla CSS, and JavaScript.

## 🚀 Features

- **Brutalist Design**: High-contrast, grid-based layout with a raw, industrial feel.
- **Pure Dark Mode**: A "pure black" dark theme toggleable via the navbar or system shell.
- **System Shell**: Interactive terminal overlay (Ctrl + K) with custom commands.
- **Responsive & Fluid**: Fully optimized for mobile, tablet, and desktop viewports.
- **Scrollytelling**: Smooth reveal animations and parallax effects.
- **Guestbook**: Anonymous message logging powered by Upstash Redis.
- **CRT Mode**: Immersive legacy tube-monitor effect via the Konami code or terminal.

## ⌨️ System Shell Commands

Press `Ctrl + K` to open the terminal. Available commands include:
- `help`: List available system commands.
- `dark`: Toggle between Light and Pure Black modes.
- `crt`: Toggle the CRT monitor effect.
- `logs`: Enable/Disable the background activity feed.
- `goto [page]`: Navigate the system (home, about, projects, etc.).
- `whoami`: Display system identity.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Typography**: JetBrains Mono, Space Mono.
- **Backend (Guestbook)**: Upstash Redis (Serverless).
- **Icons/Media**: Custom SVG icons and AI-generated brutalist assets.
